#OneKey Pacman
## Good ol' pacman game made for OneKey HackClub!
(The game is made using HTML,CSS,Js and a really big brain.)

## Features
- Fully Playable using just the Spacebar!
- Pacman is faster than the ghosts(to make it a little easy)
